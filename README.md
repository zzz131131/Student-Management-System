# 学生管理系统（C++实现）
基于C++实现的控制台版学生信息管理系统，包含完整增删改查功能，适合新手学习。

## 功能介绍
- ✅ 添加学生信息
- ✅ 显示所有学生信息
- ✅ 删除学生信息
- ✅ 查找学生信息
- ✅ 修改学生信息
- ✅ 清空学生信息
- ✅ 退出系统

## 开发环境
- 语言：C++
- IDE：Visual Studio 2022
- 项目类型：控制台应用

## 使用方法
1. 点击右上角 `Code` → `Download ZIP`
2. 解压后双击 `学生管理系统.vcxproj` 用VS打开
3. 按下 `Ctrl+F5` 直接运行

## 项目亮点
- 代码结构清晰，注释完整
- 菜单式交互，操作简单
- 纯C++实现，无第三方依赖
